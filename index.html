The Perfect App

  • Portable
  • Creates Easy to Read Diagrams
  • Easy to Learn - Small Learning Curve
  • Convenient to Use - Works across Multiple Devices and OSes
  • Relatively Cheap

VeeGeeX [V-G-X] Goals

  • Portable
    • Produce SVG Images. Usable on any Vector Graphing Program
  • Creates Easy to Read Diagrams
    • Leverages SVG which Scales to any Size Without Losing Clarity
  • Easy to Learn
    • Simple Mouse and Touch Gestures. Help Model similar to those found on popular SmartPhone OSes

VeeGeeX [V-G-X] Goals

  • Convenient to Use
    • Works in the browser. Will have IE, Firefox, Chrome Desktop and Mobile Support. Final Images can be stored in the Cloud or local to the Device
  • Relatively Cheap
    • Pre-release versions will be free to use. OpenSource version will be available with basic functionality
